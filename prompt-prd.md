

# AI 专用开发手册：萌萌动物方块 (Cute Animal Blocks)

## 1. 项目概况 (Context)
* **目标：** 开发一款网页版俄罗斯方块游戏，作为送给女朋友的礼物。
* **核心风格：** 软萌动物风（参考奥比岛 UI），强调 Q 弹的视觉反馈和治愈感。
* **技术栈：** Vue 3 (SFC), Vite, TypeScript, GSAP (动画), Canvas API (游戏渲染)。

## 2. 核心技术规范 (Technical Specs)
### A. 游戏逻辑矩阵
* **网格系统：** $10 \times 20$ 标准矩阵。
* **方块类型 (Tetrominos)：** 对应特定动物形象（见视觉说明）。
* **状态管理：** 使用 Vue 3 `reactive` 或 `Pinia` 存储 `score`, `level`, `matrix`, `nextPiece`。

### B. 渲染引擎
* 使用 **HTML5 Canvas** 进行主游戏区域渲染。
* UI 界面（分数、按钮、弹窗）使用 **Vue 组件 + Tailwind CSS**。

---

## 3. 视觉与动效指令 (The Vibe)
### A. 动物映射关系 (Asset Mapping)
AI 在编写渲染逻辑时，请为不同形状赋予对应的颜色和装饰：
* **I块 (趴趴兔):** `#FFB7C5` (樱花粉)，长条状。
* **O块 (小胖熊):** `#F3E5AB` (奶油黄)，正方形。
* **T块 (小猫咪):** `#B29DD9` (薰衣草紫)，T形。
* **L/J块 (小企鹅):** `#A7C7E7` (浅海蓝)。
* **S/Z块 (小粉猪):** `#FFDAC1` (蜜桃橘)。

### B. 关键动效 (GSAP Instructions)
* **落地回弹 (Landing):** 当方块触底锁定瞬间，执行一个 `scaleY` 从 $1.0 \to 0.9 \to 1.0$ 的弹性动画。
* **旋转反馈:** 旋转时使用平滑旋转而非瞬间切换。
* **消除动画:** 被消行的方块需向四周迸发“心形粒子”，然后渐隐。

---

## 4. 核心功能需求 (Features)
1.  **基础玩法：** 左右移动、变形、硬降 (Hard Drop)、预览下一个方块。
2.  **礼物特供：**
    * 当分数为 `520` 或 `1314` 时，触发全屏 `Confetti`（爱心纸屑）。
    * 背景音乐循环播放轻快 Lo-fi。
3.  **响应式布局：** 适配手机端全屏显示，支持触屏手势（左右滑动移动，点击旋转，下滑加速）。

---

## 5. 文件结构建议 (Architecture)
AI 请按以下结构组织代码：
* `src/engine/GameCore.ts`: 纯逻辑层（矩阵计算、碰撞检测、消行算法）。
* `src/engine/Renderer.ts`: Canvas 渲染层。
* `src/components/GameView.vue`: 主视图组件。
* `src/hooks/useControls.ts`: 键盘与手势监听。

---

## 6. 开发者纪律 (Rules for AI)
1.  **拒绝锐角：** 所有生成的 CSS 必须包含 `border-radius: 20px` 或以上。
2.  **代码质量：** 禁止编写超过 200 行的单体函数，逻辑复杂的算法需加中文注释。
3.  **类型安全：** 严格使用 TypeScript 定义方块类型和矩阵状态。
4.  **互动优先：** 如果我没提供图片素材，请先用 Canvas 画出带圆角的色块，并加上“眼睛”小点来模拟动物感。
